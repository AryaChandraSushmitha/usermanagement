# User Management System

 React application for managing user data. 

## Features

- View list of users with their details
- Add new users
- Edit existing users
- Delete users
- Loading states and error handling

## Project Setup

1. Clone the repository and navigate to the project directory:
```bash
  cd user-management
```

2. Install dependencies:
```bash
  npm install
```

3. Start the development server:
```bash
  npm start
```

The application will open in your default browser at http://localhost:3000

## Project Structure

All the components are in the src/components folder.

App.js is the main file.

UserList.js is the component for the user list.

UserForm.js is the component for the user form.

styles.css is the file for the styles.

ErrorBoundary.js is the component for the error boundary.
